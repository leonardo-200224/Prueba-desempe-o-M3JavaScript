# SpaceBook 

-SpaceBook is a Single Page Application  built with Vanilla JavaScript + Vite that allows company employees to reserve shared workspaces. Administrators can manage all reservations, approve or reject them, and have full CRUD control over the system.

The application implements authentication, role-based access control, session persistence, SPA routing via the History API, and full CRUD operations against a simulated REST API powered by json-server.

## Technologies Used

- Vite 8	
- Vanilla JavaScript 
- TailwindCSS 
- json-server	
- localStorage	
- History API 
- Fetch API	HTTP 
- concurrently	

## Installation

- npm install
- npm run dev
